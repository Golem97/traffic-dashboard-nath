# Components

React components organized by functional domain.

## Structure:
- `ui/` - Shadcn/UI components (Button, Card, Table, etc.)
- `dashboard/` - Traffic dashboard specific components
- `auth/` - Authentication components (Login, Register, etc.)
- `common/` - Reusable components across the app

## Rules:
- One component = one file
- Use only Tailwind CSS
- Export as default
- Type with TypeScript
